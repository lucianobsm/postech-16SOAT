      }

2. SERVIÇO DE APLICAÇÃO (OrdemServicoService)
   ├─ Busca OS pelo orcamentoId
   ├─ Valida status EM_DIAGNOSTICO
   ├─ Executa regra de domínio: concluirDiagnostico()
   ├─ Persiste alterações
   ├─ Registra evento em histórico
   ├─ Gera PDF via PdfGeneratorPort
   ├─ Monta template HTML
   └─ Envia E-mail via EmailSenderPort

3. ADAPTER PDF (PdfGeneratorAdapter)
   ├─ Cria documento A4 com iText
   ├─ Adiciona cabeçalho com dados da oficina
   ├─ Insere dados do orçamento e veículo
   ├─ Tabela de serviços/mão de obra
   ├─ Tabela de peças/insumos
   ├─ Resumo financeiro com valor total
   └─ Retorna byte[] do PDF

4. ADAPTER E-MAIL (EmailSenderAdapter)
   ├─ Cria mensagem MIME
   ├─ Define assunto, corpo HTML, destinatário
   ├─ Anexa PDF com nome descritivo
   └─ Envia via JavaMailSender

 SETUP LOCAL EM 3 PASSOS
═══════════════════════════════════════════════════════════════════════════════

PASSO 1: Iniciar infraestrutura
$ docker-compose -f docker-compose.dev.yml up -d
  └─ PostgreSQL: localhost:5433
  └─ MailHog: localhost:1025 (SMTP) e localhost:8025 (Web UI)

PASSO 2: Configurar variáveis
$ export SPRING_PROFILES_ACTIVE=dev
$ cp .env.example .env

PASSO 3: Executar aplicação
$ ./mvnw spring-boot:run
  └─ Acesse: http://localhost:8080
  └─ MailHog Web UI: http://localhost:8025
